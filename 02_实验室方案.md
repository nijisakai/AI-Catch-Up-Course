## 三、前部 · MeetaVista 数字助教交互区

### 设计意图
- **入门第一印象**：进门第一眼就是裸眼 3D 数字人在讲课程脉络，瞬间触发"这不是普通培训班"
- **实时教学辅助**：课中跟主讲节奏自动切换内容（脉络图 / 关键词卡片 / Checkpoint 进度）
- **课间答疑**：学员走到屏前问问题，数字人调本课程 RAG 回答
- **招生展示**：媒体来访时是 demo 主角

### 技术架构

```
   学员/讲师/外宾
        ↓ 语音 / 触控 / 视觉
   ┌──────────────────────────────────┐
   │   MeetaVista 数字助教            │
   │  [Sony ELF-SR2 裸眼 3D 显示]     │
   │  [3D 数字人渲染 (魔珐有言/Hedra)]│
   │  [LLM 控制层]                    │
   │   ├─ Claude Sonnet 4.6（主问答） │
   │   ├─ Ollama Qwen3 32B（兜底）   │
   │   └─ 本地 RAG（课程全部知识包） │
   │  [语音 ASR + TTS]                │
   │   ├─ Whisper.cpp（识别）         │
   │   └─ ElevenLabs / CosyVoice      │
   │  [视点跟踪 (Sony 内置传感器)]    │
   └──────────────────────────────────┘
```

### 数字助教知识包（启动时加载到本地 RAG）
- 完整课程大纲（72 节）
- 全部知识词条（28 篇 LLM Wiki）
- 全部讲师剧本（72 节）
- 学员物料 + 实验室方案
- 30+ Skill 卡片
- 每位在校学员的进度

### 三种工作模式

| 模式 | 触发条件 | 行为 |
|---|---|---|
| **课程模式** | 上课时间 | 跟主讲节奏切内容（不主动说话） |
| **答疑模式** | 学员走近 + 唤醒词 | 听问题 → RAG 回答 → 弹参考资源 |
| **展示模式** | 媒体 / 外宾 / 招生 | 主动播报 + 引导参观 |

## 十五、关键采购建议

1. **Sony ELF-SR2 必须国行**（保修 + 配件）；可同时给京东方 27" 国产备一个 Plan B 询价。
2. **GPU 工作站不要图便宜买二手**（5090 一手与二手差价仅 1.5 万，但二手风险大）。
3. **全息体验区按"能用就好"采购**——这块要的是体验震撼，不是性能极致；Looking Glass Go 6" 桌面版就足以让学员"哇"。
4. **NAS 必上万兆**——学员同时拉 30GB 模型时千兆会瓶颈到崩溃。
5. **保险必上**——学员误碰 Vision Pro / 全息屏的概率不低，事先买好。
