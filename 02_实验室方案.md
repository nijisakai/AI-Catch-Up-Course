# 《AI 跟上时代》实验室建设方案

> 总预算 ¥60 万一次性投入 · 80-100㎡ 三段式空间
> 默认标准班型；企业自建实验室或更高规格可按需深度扩展

---

## 项目背景

率先在企业 / 培训机构内部搭建专属 AI 数字员工训练实验室，是把 AI 能力**沉淀为组织资产**的最直接路径。本方案以**"前为戏 / 中为练 / 后为秀"**三段式空间为核心，把"招生展示 / 日常授课 / 学员体验"三大场景在一个空间里**视觉解耦、动线连贯**地完成。

设计目标：

- **第一印象足够震撼**：进门即见裸眼 3D 数字助教在讲课程脉络
- **日常授课足够稳定**：主讲席 + 学员席常规配置可承载 15-30 人
- **课间体验足够丰满**：后部全息体验区让学员真正"摸到"空间计算最新装备
- **运维成本足够低**：本地算力 + 云端 Token 混合架构，保证学员练手"无限张"，关键演示用最强模型

---

## 二、方案亮点

**1、三段式空间布局**：前为戏 / 中为练 / 后为秀，**功能解耦但视线通透**，不做硬隔断。

**2、MeetaVista 数字助教三种工作模式**：课程模式（同步切片）/ 答疑模式（学员唤醒）/ 展示模式（媒体导览），一台设备扛三个场景。

**3、本地 + 云端混合算力**：本地 RTX 5090 工作站跑 ComfyUI / 本地大模型 / 视频生成 / RAG，云端 Token 跑闭源最强模型。**学员练手"无限张"，关键演示用最强模型**。

**4、全息体验区差异化**：Vision Pro / Quest 3 / Looking Glass / 国产体感投影 / 全息金字塔 五件套 —— 普通 AI 培训班没有的视觉冲击。

**5、可观测与运维**：UPS + 防火墙 + 监控 + 万兆网络 + NAS 集中存储 —— **课件 / 模型 / 学员作品全部本地化**。

**6、企业级深度定制**：本方案为标准班型；企业自建实验室或行业专用（医疗 / 法律 / 制造）可按需扩展。

## 三、前部 · MeetaVista 数字助教

### 设计意图
- **入门第一印象**：进门第一眼是裸眼 3D 数字人在讲课程脉络，瞬间触发"这不是普通培训班"。
- **实时教学辅助**：课中跟主讲节奏自动切换内容（脉络图 / 关键词 / Checkpoint 进度）。
- **课间答疑**：学员走到屏前问问题，数字人调本课程 RAG 回答。
- **招生展示**：媒体来访时是 demo 主角。

### 系统架构

> 学员 / 讲师 / 外宾 → 语音 / 触控 / 视觉三种输入 → 经下面 5 层处理 → 裸眼 3D 数字人输出。

<!-- arch -->
| 层级 | 实现 |
|---|---|
| **输入层** | 语音 / 触控 / 视觉三通道（学员 / 讲师 / 外宾任意一方触发） |
| **显示层** | 裸眼 3D 大屏 + 3D 数字人实时渲染（魔珐有言 / Hedra 3D） |
| **LLM 控制层** | Claude Sonnet 4.6（主问答）<br>Ollama Qwen3 32B（兜底，断网可用）<br>本地 RAG（72 节课程知识包 + 学员进度） |
| **语音层** | Whisper.cpp（ASR 本地识别）<br>ElevenLabs / CosyVoice（TTS 合成） |
| **跟踪层** | 视点跟踪（Sony 内置传感器）—— 自动切换数字人朝向与互动模式 |

### 数字助教知识包（启动加载）
- 完整课程大纲（72 节）
- 全部讲师剧本 + 知识词条
- 学员物料 + 实验室方案
- 30+ Skill 卡片
- 每位在校学员的进度

### 三种工作模式

| 模式 | 触发条件 | 行为 |
|---|---|---|
| 课程模式 | 上课时间 | 跟主讲节奏切内容，不主动说话 |
| 答疑模式 | 学员走近 + 唤醒词 | 听问题 → RAG 回答 → 弹参考资源 |
| 展示模式 | 媒体 / 外宾 / 招生 | 主动播报 + 引导参观 |

### 整体造价

> **MeetaVista 数字助教整套**（裸眼 3D 大屏 + 驱动机 + 触控副屏 + 麦阵 + 监听音箱 + 中控 + 装饰外壳 + 配件 + 上电调试）：**¥150,000**

## 十三、企业版扩展建议

本方案为标准班型，**企业自建实验室或更高规格**可按以下方向扩展：

1. **裸眼 3D 主屏升级到工程级**（如 Sony ELF-SR2 27" 国行 ¥50,000 / 京东方 65" 大屏方案）。
2. **全息体验区追加 Looking Glass HLD 27" / 86"**（含关税单台 ¥75,000 起）。
3. **算力升级到双 / 四 RTX 5090 工作站**（用于行业大模型微调）。
4. **行业专用工作台**（医疗影像 / 法律合同 / 财务报表 高频场景），按行业 SOP 定制。
5. **多分会场远程互联**（万兆专线 + 多点同步）。
