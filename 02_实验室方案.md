# 《AI 跟上时代》实验室建设方案

> **总预算**：¥60 万一次性投入
> **场地建议**：80-100㎡，层高 ≥ 2.8m，单/双面采光 + 遮光帘，独立线路
> **空间逻辑**：**前为戏 / 中为练 / 后为秀** —— 三段式布局，前后两端是亮点，中间是日常授课场。
> **企业定制**：本方案为标准班型；企业自建实验室或更高规格可按需扩展（联系教务）。

## 二、前部 · MeetaVista 数字助教

### 设计意图
- **入门第一印象**：进门第一眼是裸眼 3D 数字人在讲课程脉络，瞬间触发"这不是普通培训班"
- **实时教学辅助**：课中跟主讲节奏自动切换内容（脉络图 / 关键词 / Checkpoint 进度）
- **课间答疑**：学员走到屏前问问题，数字人调本课程 RAG 回答
- **招生展示**：媒体来访时是 demo 主角

### 系统架构

```
   学员/讲师/外宾
        ↓ 语音 / 触控 / 视觉
   ┌──────────────────────────────────┐
   │   MeetaVista 数字助教            │
   │  [裸眼 3D 显示]                  │
   │  [3D 数字人渲染]                 │
   │  [LLM 控制层]                    │
   │   ├─ Claude Sonnet 4.6（主问答） │
   │   ├─ Ollama Qwen3 32B（兜底）   │
   │   └─ 本地 RAG（课程全部知识包） │
   │  [语音 ASR + TTS]                │
   │   ├─ Whisper.cpp（识别）         │
   │   └─ ElevenLabs / CosyVoice      │
   │  [视点跟踪]                      │
   └──────────────────────────────────┘
```

### 数字助教知识包（启动加载）
- 完整课程大纲（72 节）
- 全部讲师剧本 + 知识词条
- 学员物料 + 实验室方案
- 30+ Skill 卡片
- 每位在校学员的进度

### 三种工作模式

| 模式 | 触发 | 行为 |
|---|---|---|
| **课程模式** | 上课时间 | 跟主讲节奏切内容，不主动说话 |
| **答疑模式** | 学员走近 + 唤醒词 | 听问题 → RAG 回答 → 弹参考资源 |
| **展示模式** | 媒体 / 外宾 / 招生 | 主动播报 + 引导参观 |

### 整体造价

> **MeetaVista 数字助教整套（裸眼 3D 大屏 + 驱动机 + 触控副屏 + 麦阵 + 监听音箱 + 中控 + 装饰外壳 + 配件 + 上电调试）**：¥150,000
