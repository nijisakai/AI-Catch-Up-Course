# 《AI 跟上时代》课程大纲

> **课时**：72 节 · 总时长约 120 小时
> **价值观**：不让学员焦虑，但也不让学员落后。
> **默认难度**：面向 35+ 普通职场人 —— 不要求编程基础，前 9 节完全不碰代码。
> **企业版**：行业 / 技能纵深、节奏强度、产出标准均可按企业需求**深度定制**（联系教务）。

---

## 营销

> 2026 年 4 月 — **会不会用 AI，正在决定你下一份工资条上的数字。**
>
> - 那条"一个人一天拍完一支广告片"的短视频——他不是天才，是会**把 AI 串起来干活**。
> - 那个天天 5 点准时下班、周报却比你漂亮的同事——他不是效率高，是**有一套 AI 帮他干杂活**。
> - 招聘启事里那行"会用 AI 工具者优先"——已经从"加分项"变成了"**门槛**"。
> - 你十年练熟的 PS / Excel / 剪映 / 写脚本，正在被新一代 AI 工具**按秒换掉**。
>
> 焦虑是正常的。**但焦虑不会让你学到任何东西，跟完这 72 节课才会。**

---

## 课程是什么

> 用 72 节课，把日常工作里能立刻派上用场的那一整条 AI 工具链 —— **从聊天到造工具、从图像到视频、从本地模型到 24×7 数字员工** —— 一节一节装到你手里，串成一条让你**在自己岗位上比同事快一个身位**的生产线。
>
> 不堆概念、不啃论文、不背术语。每节课结束你都能掏出一件能用的东西回家。

学完后你**真的做得出来**：

|---|---|---|

# M0 · 电脑基本功 + AI 工作站（14 节，必看）

> 17 节压缩为 14 节——把"硬件选购 / 文件整理 / 密码安全 / 截图录屏 / 浏览器 / 桌面客户端 / 提示词 / GitHub 速读 / 编辑器 / 终端 / Node / Python / 代理 / API Key" 全部覆盖。
> **前 9 节完全不碰代码**——零基础也能进。

### M0-1 · 为什么 2026 学 AI 要重新认识你的电脑

- **学习目标**：理解"开发环境 = 工作台"，搞清浏览器 / 桌面 / 编辑器 / 终端四件套。
- **生活化场景**：早上打开电脑发呆 → 接下来每个 AI 工具到底放哪、点哪？
- **配套 Skill**：[`system-check`](https://github.com/anthropics/skills) —— 30 秒一键自检硬件 / OS / 网络。
- **YouTube（真实）**：[Anthropic · AI prompt engineering: A deep dive](https://www.youtube.com/watch?v=T9aRN5JkmL8)（先看这个建立 AI 沟通直觉，工作站设置类视频可参考 NetworkChuck 频道）。
- **资料**：自检脚本 `system-check.ps1` / `system-check.sh`。

### M0-2 · 硬件选型：3K-50K 的 AI 电脑怎么配

- **学习目标**：下次换电脑你就是最懂 AI 场景的买家。
- **生活化场景**：女朋友 / 老婆问"你为啥又要换电脑？" 你能 5 分钟讲清。
- **配套 Skill**：[`pc-builder-advisor`](https://github.com/karanb192/awesome-claude-skills) —— 输入预算 + 用途，输出 3 套配置 + 京东直链。
- **YouTube（参考频道）**：David Ondrej / Matthew Berman 频道有大量"硬件选型"主题（搜索"AI PC build 2026"）。
- **坑点**：8GB 内存的 Mac 别买；M4 16GB 起步；AMD 显卡 AI 生态目前仍有坑。

### M0-3 · 文件管理与命名：让"乱糟糟桌面"消失

- **学习目标**：学会"文件 = 资产"思维，落地三层结构（工作 / 资料 / 归档）。
- **生活化场景**：客户问"你上周给我的那版方案在哪？"——10 秒找到。
- **配套 Skill**：[`file-organizer`](https://github.com/VoltAgent/awesome-agent-skills) —— Claude 自动按语义重命名 + 归档旧文件。
- **YouTube（参考频道）**：Riley Brown 频道有"AI-first filing"系列（@rileybrown_ai 搜"file organization AI"）。
- **资料**：[Everything](https://www.voidtools.com/) / [Raycast](https://www.raycast.com/)。

### M0-4 · 密码与账号安全：AI 时代账号 = 货币

- **学习目标**：会用密码管理器、会开 2FA、会识别钓鱼。
- **生活化场景**：API Key 一不小心 push 到 GitHub，3 小时被扣 800 美金 —— 真实案例。
- **配套 Skill**：[`secret-scanner`](https://github.com/anthropics/skills) —— 提交前自动扫敏感词 + .env 钩子。
- **YouTube（参考频道）**：The PC Security Channel / Lawrence Systems 频道有 AI 钓鱼实测系列。
- **坑点**：短信 2FA 不安全（SIM swap）；能用 TOTP 就别用短信。

### M0-5 · 截图、录屏、语音输入：喂给 AI 的三把刷子

- **学习目标**：30 秒内把任何东西塞给 AI 的肌肉记忆。
- **生活化场景**：Excel 报错截图丢给 Claude → 3 秒看出来是公式哪里写错。
- **配套 Skill**：[`vision-helper`](https://github.com/anthropics/skills) —— 截图 → 自动 OCR + 描述 + 改写需求。
- **YouTube（参考频道）**：AI Jason（@AIJasonZ）频道有大量"voice-first workflow"实操。
- **工具**：Snipaste / [SuperWhisper](https://superwhisper.com/) / [Wispr Flow](https://wisprflow.ai/)。

### M0-6 · 浏览器 2.0：从 Chrome 到 Dia / Arc / Comet

- **学习目标**：理解"浏览器 = AI 入口"。
- **生活化场景**：边看英文论文边中文对照 + 一键问 Claude。
- **配套 Skill**：[`web-snapshot`](https://github.com/ComposioHQ/awesome-claude-skills) —— 把整页 HTML+ 截图一键存 Markdown 喂 AI。
- **YouTube（参考频道）**：Matt Wolfe（@mreflow）每周新工具速评有 Dia / Comet / Arc 对比。
- **必装扩展**：沉浸式翻译 / Monica / ChatHub。

### M0-7 · AI 桌面客户端全家桶：一次装齐

- **学习目标**：Dock / 任务栏应该常驻哪些 AI 客户端。
- **生活化场景**：3 秒唤起 Claude，跟"打开微信"一样自然。
- **配套 Skill**：[`raycast-launcher-pack`](https://github.com/raycast/extensions) —— 13 个 AI 高频快捷键预设。
- **YouTube（参考频道）**：Jeff Su（@JeffSuOfficial）频道有"AI for Professionals"完整系列。
- **客户端**：[Claude Desktop](https://claude.ai/download) + [ChatGPT Desktop](https://openai.com/chatgpt/download) + [Raycast](https://www.raycast.com/) / PowerToys。

### M0-8 · 提示词基础：先把话说清楚

- **学习目标**：在碰任何 AI 工具前，先学会"怎么问 AI"。
- **生活化场景**：把"帮我写周报"改成"角色 + 上下文 + 任务 + 格式"四要素法 → 质量翻倍。
- **配套 Skill**：[`prompt-improver`](https://github.com/anthropics/prompt-eng-interactive-tutorial) —— 输入烂提示词 → 输出 4 要素改写版。
- **YouTube（真实）**：[Anthropic · AI prompt engineering: A deep dive](https://www.youtube.com/watch?v=T9aRN5JkmL8) + [Karpathy · Software Is Changing (Again)](https://www.youtube.com/watch?v=LCEmiRjPEtQ)。
- **坑点**：2026 年模型已经聪明，把事情讲清楚 80% 就够了，别追"魔法咒语"。

### M0-9 · 看懂一个 GitHub 仓库（不写一行代码）

- **学习目标**：30 秒判断"值不值得装、能不能用"。
- **生活化场景**：朋友圈刷到一个 GitHub 项目，3 分钟告诉自己要不要折腾。
- **配套 Skill**：[`repo-summarizer`](https://github.com/karanb192/awesome-claude-skills) —— 输入仓库 URL → README 摘要 + 安装难度评分 + 是否商用。
- **YouTube（真实）**：[AI Turns ANY GitHub Repo into an EASY Tutorial (FREE & Open Source)](https://www.youtube.com/watch?v=wJ42uIVGFA4)。
- **工具**：[gitingest](https://gitingest.com/) / [DeepWiki](https://deepwiki.com/) / [GitHub Trending](https://github.com/trending)。

# M1 · AI 底层逻辑速通（5 节，必看）

### M1-1 · AI 演进三幕：Prompt → Context → Harness

- **学习目标**：用一节课讲清 2022-2026 AI 工程范式三次迁移。
- **生活化场景**：派新人去客户拜访 — Prompt 是把任务讲清楚，Context 是把资料备齐全，Harness 是带 checklist + 实时汇报 + 验收。
- **配套 Skill**：[`harness-self-check`](https://github.com/anthropics/skills) —— 给任意 Agent 任务做"上下文 / 工具 / 编排 / 记忆 / 评估 / 校验"六层体检。

- **配套 Skill**：[`context-compactor`](https://github.com/VoltAgent/awesome-agent-skills) —— 长对话自动压缩 + 关键信息提取。
- **YouTube（真实）**：[Matthew Berman · Manus - The ALL-IN-ONE AI AGENT](https://www.youtube.com/watch?v=wVvfXZm4Lvg)（讲了 Manus 的 context 设计）。

### M1-4 · MCP 协议：让 LLM 真正"动手"的总线
