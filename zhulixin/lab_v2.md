# **一、建设目标与定位**

本实验室面向「企业高管」与「大学非计算机专业学生」两大核心群体，打造国内领先的
AI 通识教育体验中心：\
\
【对企业高管】\
• 提供零代码/低代码的 AI
工具链体验，聚焦商业汇报（PPT/文档自动化）、战略决策辅助、办公效率提升\
• 通过 MeetaVista 数字人、全息投影等前沿设备，直观感受 AI
对企业文化展示、客户接待、品牌传播的变革价值\
• 配备 WorkBuddy、Coze、OpenClaw 等企业级 Agent 平台，模拟真实办公场景\
\
【对大学生】\
• 消除技术恐惧，通过\"会说话就能编程\"的 AI 编程工具（Claude Code /
OpenCode / DeepSeek-TUI），让非计算机专业学生也能开发实用小工具\
• 结合学术场景（文献管理、论文写作、数据分析），打造\"AI +
专业\"的跨学科创新空间\
• 通过 AI 教育机器人、AR 投影等硬件，激发学习兴趣与创造力\
\
【空间定位】\
三段式空间设计：前部数字助教区 + 中部教学实操区 + 后部全息体验区

**A.1 MeetaVista 裸眼 3D 数字人系统**技术原理：光栅裸眼 3D 显示 +

触控\

≥ 95%\
• 知识库：支持 RAG 检索增强，本地部署可处理 10 万+ 文档\
\
供应商参考：\
• 国内：MeetaVista、硅基智能、小冰公司、魔珐科技\
• 国际：Soul Machines、Didimo、Synthesia（需评估合规性）\
\
部署建议：\

• 建议搭配定向音场，避免不同课程间声音干扰\

1. 安装 Node.js 18+：curl -fsSL https://deb.nodesource.com/setup_18.x \|
sudo -E bash -\
2. 安装 Claude Code：npm install -g \@anthropic-ai/claude-code\
3. 登录 Anthropic 账号：claude login\
4. 验证安装：claude \--version\
\
配置优化（中国网络环境）：\
• 设置代理：export HTTPS_PROXY=http://127.0.0.1:7890\
• 或使用国内镜像：claude config set api-base-url
https://api.kimi.com/coze（如需接入国产模型）\
\
【OpenCode 安装】\
系统要求：Node.js 18+\
安装步骤：\
1. npm install -g opencode-ai\
2. 配置模型提供商：opencode config add-provider deepseek \--api-key
YOUR_KEY\
3. 启动：opencode chat\
\
支持模型（75+ 提供商）：\
• 国产：DeepSeek V4 / Kimi k2.5 / 通义千问 3 / 文心一言 4 / 讯飞星火 4\
• 国际：Claude 3.7 / GPT-4o / Gemini 2.5 / Llama 3\
• 本地：Ollama（Llama / Mistral / 通义千问本地版）\
\
【DeepSeek-TUI 安装】\
系统要求：Python 3.9+\
安装步骤：\
1. pip install deepseek-tui\
2. 配置 API Key：deepseek-tui config set api_key YOUR_KEY\
3. 启动：deepseek-tui \--model deepseek-chat\
\
特色功能：\
• 1M 长上下文：支持上传整本 PDF（如 500 页教材）进行全文分析\
• 文件拖拽：直接将文件拖入终端窗口即可上传\
• 多会话管理：支持创建多个独立会话，上下文隔离

**B.2 Agent 平台部署**【Coze（扣子）------ 零代码平台】\
• 访问 coze.cn，字节账号登录\
• 创建 Bot → 选择\"工作流\"模式 → 拖拽节点\
• 内置插件：天气、股票、翻译、地图、电商价格监控等 100+\
• 知识库：上传 Word/PDF/网页，支持自动分段与向量化\
• 发布渠道：抖音、飞书、微信、微信公众号\
\
【OpenClaw ------ 开源框架】\
• 安装：npm install -g openclaw\
• 配置：openclaw config set-model deepseek-chat\
• 核心概念：\
- Agent：单个智能体，有角色设定、工具集、记忆系统\
- TaskFlow：任务流，编排多个 Agent 的执行顺序与条件分支\
- REM：记忆系统，自动记录对话历史，支持跨会话召回\
• 示例：创建\"每日新闻摘要\"TaskFlow\
openclaw task create \--name \"news_digest\" \\\
\--trigger \"cron:0 8 \* \* \*\" \\\
\--steps \"fetch_news-\>summarize-\>send_email\"\
\
【WorkBuddy ------ 桌面智能体】\
• 官网下载安装包（Windows/Mac）\
• 微信扫码登录\
• 核心能力：\
- 文件操作：\"把桌面所有 JPG 图片重命名为\'照片_日期_序号\'\"\
- Office 自动化：\"根据这份 Excel 数据生成 PPT，每页一个产品\"\
- 微信操控：在微信对话框发送\"帮我整理今天的会议纪要\"\
• 企业版：支持私有化部署、企业微信集成、权限管理

# **附录 C：安全与合规指南**

【数据安全】\
• 高管学员商业数据：建议本地部署大模型（Ollama +
私有知识库），禁止上传公有云\
•
学生学员个人信息：符合《个人信息保护法》，人脸/声纹数据本地存储，不上传第三方\
• 实验数据隔离：每位学员独立工作区，课程结束后可选择导出或删除\
\
【内容合规】\
• AI 生成内容标识：所有 AI 生成的 PPT/论文/图片需标注\"AI 辅助生成\"\
• 学术伦理：学生使用 AI 辅助论文写作需声明使用范围，禁止直接提交 AI
生成内容\
• 版权风险：M3-8 课程专门讲解 AI 生成内容的版权归属与使用边界\
\
【设备安全】\
• 全息展柜：避免直视强光源，设置自动休眠（无人操作 5 分钟后关闭）\
• 教育机器人：设置碰撞检测与急停按钮，避免学员受伤\
• 3D 扫描仪：扫描人体时需获得明确同意，禁止扫描敏感部位
