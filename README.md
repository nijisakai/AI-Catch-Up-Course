## 设计原则

1. **不分级**：不再 🌱🌿🌳，**只分"必看 / 选看"**——焦虑的人最怕"学错了"。
2. **每节配 1 个 Skill**：让"听完就有装备"成为肌肉记忆。
3. **每节配 1 个真实 YouTube 一手视频**：不二手翻译，直接看 Karpathy / Matthew Berman / Fireship 等原片。
4. **每节配 1 个生活化场景**：邮件三筛、合同高亮、会议提取、报销对账等——35+ 学员一看就觉得"这是我每天的事"。
