## 设计原则（v3 比 v1/v2 更明确）

1. **不分级**：不再 🌱🌿🌳，**只分"必看 / 选看"**——焦虑的人最怕"我学的是不是不该学的"。
2. **每节配 1 个 Skill**：让"听完就有装备"成为肌肉记忆。Skill 来自 Anthropic 官方 + Composio + VoltAgent + Awesome-Claude-Skills 等公开仓库。
3. **每节配 1 个真实 YouTube 一手视频**：26 节都标了具体 URL（[详见视频清单](05_视频资源/README.md)），不二手翻译。
4. **每节配 1 个生活化场景**：邮件三筛、合同高亮、会议提取、表格补齐、报销对账等——35+ 学员一看就觉得"这是我每天的事"。
5. **价格全部 WebSearch 实时核实**：详见 [02_实验室方案.md](02_实验室方案.md) 的"价格变化对比表"，v2 估算与真实价的差异都列出来了。
